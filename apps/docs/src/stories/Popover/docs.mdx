    <p>Стандартный поповер с формой входа. Открывается по клику на кнопку. Требует указания элемента-триггера через свойство <code>rootEl</code>.</p>

    <p>Поповер может открываться по клику (<code>trigger="click"</code>) или при наведении (<code>trigger="hover"</code>). Для hover-триггера можно настроить задержки через свойства <code>showDelay</code> и <code>hideDelay</code>.</p>

    ## Стрелка

    <p>Поповер может отображаться со стрелкой, указывающей на элемент-триггер, или без неё. Управляется свойством <code>arrow</code>.</p>

    <p>Использование слота <code>trigger</code> для создания кастомного элемента-триггера. Позволяет полностью контролировать внешний вид и поведение триггера.</p>
